# 调试指南 - 颜色格式化问题

## 问题：点击确认后没有应用颜色

如果点击确认按钮后，文本颜色和背景颜色没有改变，请按照以下步骤调试：

## 调试步骤

### 1. 查看浏览器控制台

在 Word 中打开任务窗格后：

1. **打开 Safari**
2. **Safari > 开发 > [你的Mac] > Microsoft Word**
3. **查看控制台标签页**

### 2. 检查日志输出

应该看到以下日志：

```
开始应用格式化...
收集到的规则: [...]
第一步：恢复全文原始颜色...
✓ 已恢复全文原始颜色
第二步：应用新规则...
处理规则: 搜索 "xxx", 字体颜色: #xxxxxx, 背景颜色: #xxxxxx
找到 X 个匹配结果
  设置字体颜色: #xxxxxx
  设置背景颜色: #xxxxxx
✓ 已应用所有规则
✓ 格式化应用成功
```

### 3. 检查关键信息

#### 检查规则是否正确收集

在控制台中输入：
```javascript
collectRules()
```

应该返回规则数组，例如：
```javascript
[
  {
    searchText: "测试",
    fontColor: "#ff0000",
    bgColor: "#ffff00"
  }
]
```

#### 检查搜索是否找到文本

查看日志中是否有：
- `找到 X 个匹配结果` - 如果 X 为 0，说明没有找到匹配的文本
- `警告: 未找到匹配 "xxx" 的文本` - 说明搜索文本不存在

#### 检查颜色值格式

规则中的颜色值应该是：
- 格式：`#RRGGBB`（例如 `#FF0000` 表示红色）
- 字体颜色：不能为空
- 背景颜色：如果是白色 `#FFFFFF`，会被设置为 null（移除高亮）

### 4. 常见问题

#### 问题 1: 找到 0 个匹配结果

**原因：** 搜索文本在文档中不存在

**解决：**
- 检查搜索文本是否正确
- 检查文档中是否确实包含该文本
- 尝试使用部分文本搜索

#### 问题 2: 规则数组为空

**原因：** 没有输入搜索文本

**解决：**
- 确保至少有一条规则包含搜索文本
- 检查输入框是否有值

#### 问题 3: 颜色格式错误

**原因：** 颜色值格式不正确

**解决：**
- 确保颜色值是 `#RRGGBB` 格式
- 检查颜色选择器是否正常工作

#### 问题 4: Office.js 错误

**原因：** Word API 调用失败

**解决：**
- 查看控制台中的错误信息
- 确保在 Word 环境中运行（不是在浏览器中）

### 5. 手动测试

在控制台中手动测试：

```javascript
// 1. 测试规则收集
const rules = collectRules();
console.log('规则:', rules);

// 2. 测试 Word API（需要在 Word 环境中）
Word.run(async (context) => {
    const body = context.document.body;
    const results = body.search("测试", {
        matchCase: false,
        matchWholeWord: false,
        matchWildcards: false
    });
    context.load(results, 'items');
    await context.sync();
    console.log('找到:', results.items.length, '个结果');
    
    if (results.items.length > 0) {
        const range = results.items[0];
        range.font.color = '#FF0000';
        range.font.highlightColor = '#FFFF00';
        await context.sync();
        console.log('已应用颜色');
    }
});
```

### 6. 检查 Word 文档

1. **确认文档中有文本**
   - 文档不能为空
   - 搜索文本必须存在

2. **检查文档保护**
   - 文档不能是只读的
   - 不能有格式保护

3. **尝试简单测试**
   - 在文档中输入 "测试"
   - 添加规则搜索 "测试"
   - 应用格式化
   - 检查是否生效
