# 快速开始指南

## 5 分钟快速启动

### 步骤 1: 安装依赖（1 分钟）

```bash
cd word-addin
npm install
```

### 步骤 2: 生成 SSL 证书（1 分钟）

```bash
./generate-cert.sh
```

然后在 macOS 的 Keychain Access 中信任证书（见 INSTALL.md）

### 步骤 3: 启动服务器（30 秒）

```bash
npm start
```

服务器将在 `https://localhost:3000` 启动。

### 步骤 4: 在 Word 中加载插件（2 分钟）

1. 打开 Microsoft Word for Mac
2. **插入** > **加载项** > **我的加载项**
3. 点击 **上传我的加载项**
4. 选择 `manifest.xml` 文件
5. 如果提示安全警告，选择 **信任并加载**

### 步骤 5: 使用插件（30 秒）

1. 在 Word 的 **开始** 选项卡中，找到 **文本格式化工具** 组
2. 点击 **显示文本格式化工具** 按钮
3. 或者使用快捷键 `Shift+Alt+O+P`（需要任务窗格已打开）

## 使用示例

### 示例 1: 格式化特定关键词

1. 在第一个输入框输入 "重要"
2. 选择红色作为字体颜色
3. 选择黄色作为背景颜色
4. 点击 **确认**

文档中所有包含 "重要" 的文本将被格式化为红色字体、黄色背景。

### 示例 2: 多条规则

1. 添加第一条规则：
   - 搜索文本: "错误"
   - 字体颜色: 红色
   - 背景颜色: 浅红色

2. 点击 **+** 按钮添加第二条规则：
   - 搜索文本: "成功"
   - 字体颜色: 绿色
   - 背景颜色: 浅绿色

3. 点击 **确认**

文档中所有 "错误" 和 "成功" 的文本将分别应用不同的颜色格式。

## 常见问题

**Q: 服务器启动失败？**  
A: 检查 3000 端口是否被占用，或确保已生成 SSL 证书。

**Q: 插件无法加载？**  
A: 确保服务器正在运行，并且已在 Keychain Access 中信任证书。

**Q: 快捷键不工作？**  
A: Office Add-in 的快捷键支持有限，建议使用功能区按钮。

**Q: 格式没有应用？**  
A: 检查搜索文本是否正确，确保文档中有匹配的文本。
