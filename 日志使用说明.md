# 日志功能使用说明

## 功能说明

插件现在支持将日志输出到本地文件，方便定位问题。

## 如何使用

### 1. 下载日志文件

在任务窗格底部，点击 **下载日志** 按钮，会自动下载一个日志文件。

日志文件命名格式：`word-addin-log-YYYY-MM-DDTHH-MM-SS.txt`

### 2. 查看日志内容

日志文件包含：
- 时间戳
- 日志级别（INFO、WARN、ERROR）
- 日志消息
- 详细数据（如果有）

### 3. 日志包含的信息

日志会记录：
- 插件初始化过程
- 按钮点击事件
- Word API 调用
- 规则处理过程
- 错误信息（包括堆栈跟踪）
- 颜色设置操作
- 搜索和匹配结果

## 故障排查

### 如果点击确认没有反应

1. **点击"下载日志"按钮**
2. **打开日志文件**
3. **查找以下关键信息**：
   - `开始应用格式化...` - 确认函数被调用
   - `收集到的规则` - 检查规则是否正确
   - `找到 X 个匹配结果` - 如果为 0，说明搜索文本不存在
   - `设置字体颜色` - 确认颜色设置操作
   - 任何 `ERROR` 级别的日志

### 如果重置功能失效

1. **下载日志文件**
2. **查找**：
   - `开始重置...` - 确认函数被调用
   - `恢复文本颜色` - 确认恢复操作
   - `清空规则` - 确认规则清空操作
   - 任何错误信息

## 注意事项

- 日志会保存在内存中，刷新页面后会清空
- 建议在操作后立即下载日志
- 日志文件可能较大，包含详细的调试信息
- 如果问题持续，请提供完整的日志文件

## 自动日志

所有操作都会自动记录日志，无需手动开启。

## 清空日志

如果需要清空日志（释放内存），可以在控制台中执行：

```javascript
logger.clear()
```
