# 快捷键说明

## Mac 键盘对应

快捷键 `Shift+Alt+O+P` 在 Mac 键盘上对应：

- **Shift** = Shift 键（⇧）
- **Alt** = Option 键（⌥）- Mac 上的 Alt 键就是 Option 键
- **O** = O 键
- **P** = P 键

### 使用方法

1. **按住** `Shift` + `Option`（⌥）
2. **按下** `O` 键
3. **松开所有键**
4. **按下** `P` 键

或者更简单的方式：
- 按住 `Shift` + `Option`，依次按 `O` 然后 `P`

## 键盘布局参考

```

┌─────────────────────────────────┐
│  ⇧ Shift    ⌥ Option    ⌘ Command │
```

## 注意事项

### Office Add-in 限制

⚠️ **重要**：由于 Office Add-in 的技术限制，快捷键有以下限制：

1. **需要任务窗格已打开**：快捷键只能在任务窗格获得焦点时工作
2. **不是全局快捷键**：不能在 Word 文档编辑时直接触发
3. **需要焦点**：任务窗格窗口必须处于活动状态

### 推荐使用方式

由于快捷键的限制，推荐以下使用方式：

1. **使用功能区按钮**（最可靠）
   - 在 Word 的 **开始** 选项卡中
   - 点击 **显示文本格式化工具** 按钮

2. **固定任务窗格**
   - 打开任务窗格后，可以将其固定在 Word 窗口一侧
   - 这样任务窗格始终可见，快捷键更容易触发

3. **使用 macOS 全局快捷键工具**（高级用户）
   - 使用 Keyboard Maestro、BetterTouchTool 等工具
   - 设置全局快捷键来打开任务窗格

## 故障排除

### 快捷键不工作？

1. **检查任务窗格是否打开**
   - 确保任务窗格已经打开并可见

2. **检查焦点**
   - 点击任务窗格窗口，确保它获得焦点

3. **检查按键**
   - 确保使用的是 `Option` 键（不是 `Command` 键）
   - Mac 上 `Alt` 键就是 `Option` 键

4. **尝试功能区按钮**
   - 如果快捷键不工作，使用功能区按钮作为替代

### 按键顺序

正确的按键顺序：
1. 按住 `Shift` + `Option`
2. 按 `O`
3. 松开所有键
4. 在 2 秒内按 `P`

## 技术实现

快捷键通过 JavaScript 键盘事件监听实现：

```javascript
// 监听 Shift + Option + O，然后等待 P
if (e.shiftKey && e.altKey && e.key === 'o') {
    // 等待下一个按键 'p'
    // 在 2 秒内按 P 即可触发
}
```

## 替代方案

如果快捷键无法满足需求，可以考虑：

1. **使用功能区按钮**（推荐）
2. **使用 macOS 系统快捷键工具**
3. **将任务窗格固定**，随时可用
