# 用户安装指南

本指南适用于非开发人员安装和使用 Word 文本颜色格式化插件。

## 前提条件

- Microsoft Word for Mac（2016 或更高版本）
- 互联网连接（如果插件部署在 Web 服务器上）

## 安装步骤

### 方法一：从安装包安装（推荐）

1. **获取安装包**
   - 从管理员或分发渠道获取 `word-addin-install.zip` 文件
   - 解压文件，找到 `manifest.xml` 文件

2. **打开 Microsoft Word**

3. **加载插件**
   - 点击顶部菜单栏的 **插入**
   - 选择 **加载项** > **我的加载项**
   - 点击 **上传我的加载项** 按钮
   - 浏览并选择 `manifest.xml` 文件
   - 如果出现安全提示，选择 **信任并加载**

4. **完成安装**
   - 安装完成后，在 Word 的 **开始** 选项卡中会出现 **文本格式化工具** 按钮组

### 方法二：从 Web 服务器安装

如果你的管理员提供了 Web 服务器地址：

1. **打开 Microsoft Word**

2. **加载插件**
   - 点击顶部菜单栏的 **插入**
   - 选择 **加载项** > **我的加载项**
   - 点击 **从 URL 添加**（如果有此选项）
   - 输入管理员提供的 manifest.xml URL
   - 或者下载 manifest.xml 文件后使用方法一

## 使用插件

### 打开任务窗格

有两种方式：

1. **使用功能区按钮**（推荐）
   - 在 Word 的 **开始** 选项卡中
   - 找到 **文本格式化工具** 组
   - 点击 **显示文本格式化工具** 按钮

2. **使用快捷键**
   - 按 `Shift + Alt + O`，然后按 `P`
   - 注意：快捷键需要在任务窗格已打开时才能使用

### 添加格式化规则

1. **输入搜索文本**
   - 在第一个输入框中输入要搜索的文本（例如："重要"）

2. **选择颜色**
   - 点击第二个颜色选择器，选择字体颜色
   - 点击第三个颜色选择器，选择背景颜色

3. **添加更多规则**（可选）
   - 点击 **+** 按钮可以添加新的规则行
   - 重复步骤 1-2 配置新规则

### 应用格式化

1. 配置完所有规则后，点击底部的 **确认** 按钮
2. 插件会在文档中搜索所有匹配的文本
3. 匹配的文本会被应用相应的颜色格式
4. 完成后会显示成功提示

### 取消操作

- 点击 **取消** 按钮可以清空所有规则并重新开始

## 使用示例

### 示例 1: 高亮重要信息

1. 搜索文本：`重要`
2. 字体颜色：红色
3. 背景颜色：黄色
4. 点击确认

文档中所有包含"重要"的文本会变成红色字体、黄色背景。

### 示例 2: 标记错误和成功

**第一条规则：**
- 搜索文本：`错误`
- 字体颜色：红色
- 背景颜色：浅红色

点击 **+** 添加第二条规则：
- 搜索文本：`成功`
- 字体颜色：绿色
- 背景颜色：浅绿色

点击确认，文档中的"错误"和"成功"会分别应用不同的颜色。

## 常见问题

### Q: 插件按钮在哪里？

A: 在 Word 的 **开始** 选项卡中，查找 **文本格式化工具** 组。如果看不到，可能需要重新加载插件。

### Q: 点击按钮没有反应？

A: 
1. 检查是否已正确安装插件
2. 如果插件部署在 Web 服务器上，检查网络连接
3. 尝试重新加载插件

### Q: 格式没有应用？

A: 
1. 检查搜索文本是否正确输入
2. 确保文档中确实存在匹配的文本
3. 检查文本是否在受保护的区域（如页眉页脚）

### Q: 如何卸载插件？

A: 
1. 打开 **插入** > **加载项** > **我的加载项**
2. 找到已安装的插件
3. 点击右侧的 **...** 菜单
4. 选择 **删除** 或 **卸载**

### Q: 插件无法加载？

A: 
1. 确保使用的是 Microsoft Word for Mac 2016 或更高版本
2. 检查网络连接（如果插件部署在 Web 服务器上）
3. 联系管理员获取帮助

## 获取帮助

如果遇到问题，请联系：

## 更新插件

如果插件部署在 Web 服务器上，更新是自动的，无需重新安装。

如果使用本地安装包，管理员会提供新版本的安装包，按照安装步骤重新安装即可。
